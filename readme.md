# Notes App

Created during a Udemy course

Creating new Accounts
Managing Notes

Tech used: Meteor, ReactJS, SCSS

## Getting Started

You must have Meteor installed on your machine. Then you can clone the repo and run:

```
meteor npm install
```

```
meteor
```

## Running the Tests

To run the tests on your machine:

```
meteor npm test
```
